# Niranjan Vijayan - Disability Justice Advocate Website (HTML Version)

A complete HTML, CSS, and JavaScript website for Niranjan Vijayan, featuring his work as a Disability Justice Advocate, Mindfulness Practitioner, and Inclusive Publisher.

## Features

- **Pure HTML/CSS/JavaScript**: No build process required
- **WCAG 2.1 AA Compliant**: Built with accessibility as a priority
- **Responsive Design**: Works seamlessly on all devices
- **Blue & White Theme**: Beautiful design with Buddha line art background
- **Keyboard Navigation**: Fully accessible via keyboard
- **Screen Reader Friendly**: ARIA labels and semantic structure

## Getting Started

1. **Open the website**: Simply open `index.html` in your web browser
   - Double-click `index.html` or
   - Right-click and select "Open with" → your preferred browser

2. **Local Server (Recommended)**: For best results, use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Python 2
   python -m SimpleHTTPServer 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```
   Then visit `http://localhost:8000` in your browser
